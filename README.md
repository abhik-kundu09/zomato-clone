## ✨ Features

- 🎥 **Full-screen video hero** with dark overlay and fade-in animation
- 🔴 **Zomato-themed UI** — red accent, dark background, glassmorphism navbar
- 📱 **Fully responsive** — works on mobile, tablet and desktop
- 🍔 **Animated hamburger menu** for mobile navigation
- 🔐 **Login & Signup modals** — smooth pop-up with switch between forms
- 📬 **Contact form section** — with subject dropdown and success state
- 🏪 **App store buttons** — linked to real Google Play & App Store pages
- 🦶 **Professional footer** — links grid, social icons, centered copyright
- ✨ **Micro-animations** — glowing hover effects, scroll transitions

## 🚀 Getting Started

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/abhik-kundu09/zomato-clone.git

# 2. Navigate into the folder
cd zomato-clone

# 3. Open in browser
# Just open index.html in any browser — no build step needed!
```

### Deploy to Vercel

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
vercel
```
